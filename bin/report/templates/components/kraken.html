Direct taxonomic read classification (Kraken/Braken)

Kraken is a bioinformatics tool that classifies sequencing reads against a reference nucleotide database (k2_core_nt_20250609) to assign taxonomic labels.
Pytaxonkit is subsequently used to retrieve the full taxonomic lineage for each taxid identified by Kraken, and the NCBI taxonomy is used to determine which taxa are viruses/viroids.
The table below shows the number and per cent of quality filtered reads assigned to each taxon identified by Kraken.
It can be sorted by alphabetical or numerical order by clicking on the column headers, and the number of entries shown can be adjusted using the dropdown menu above the table.
The results can be limited to a specific taxon by typing its name in the search box (for example "virus").
The table can also be filtered to only show rows which meet minimum coverage criteria and default term criteria.
The pie chart on the right provide a visual summary of the taxonomic composition at the kingdom level (top). The top 10 most abundant taxa detected within a kingdom can be displayed by clicking on the desired kingdom and these can be viewed at the family, genus or species level (bottom).

The "Matches keywords" column indicates whether the taxon name matches a specified list of terms.
