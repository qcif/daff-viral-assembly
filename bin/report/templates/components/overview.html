This report contains tables and plots to help interpret the outcome of the sequencing run and the viral identification analyses performed. Section I summarises the input data and quality control metrics. Section II provides an overview of the virus identified by 1) megablast homology search using de novo assembled contigs and/or 2) direct read search (using Kraken2 and Kaiju). Section III-V show the detailed results for each of these analyses separately (Kraken, Kaiju2 and megablast). Section VI shows consensus generated when mapping back to viral reference genomes identified by megablast in Section III. Section VII shows potential novel viral sequences identified by de novo assembly that did not match any known viral reference sequence but have viral characteristics (based on geNomad and hmmscan).
