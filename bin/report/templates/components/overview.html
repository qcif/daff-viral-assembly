This report contains tables and plots to help interpret the outcome of the RNASeq run and the viral detection analyses performed.
Section I provides a summary of the input parameters, quality control metrics and an overview of the viruses/viroids detected by both BLAST homology and direct read homology searches (provided that at least one de novo contig was assigned to these viruses/viroids).
Section II and III provides the detailed results for the direct read search performed using Kraken2 and Kaiju respectively.
Section IV shows the candidate virus detected using de novo assembly followed by megablast homology search. After mapping reads back to the contigs obtained, coverage statistics are also derived.
Section V shows consensus generated when mapping back to viral reference genomes detected by megablast in Section IV.
Section VI shows potential novel viral sequences derived by de novo assembly that did not match any known viral reference sequence but have viral characteristics (based on geNomad and hmmscan).
